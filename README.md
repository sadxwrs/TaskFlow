# TaskFlow — Локальный трекер задач

> MVP веб-приложения для управления задачами.
> Реализован на чистом HTML/CSS/JavaScript — без фреймворков, без сервера, без регистрации.

---

## О проекте

**TaskFlow** помогает справляться с хаосом в повседневных задачах для индивидуальных пользователей и мини-команд. Это приложение предлагает удобный список задач в браузере с акцентом на простоту и автономность.
Ключевые особенности:

**Основные преимущества:**
- 🛡️ Абсолютная конфиденциальность — данные остаются в вашем браузере (localStorage)
- 📴 Оффлайн-режим — работает без сети
- 🚀 Быстрый запуск — просто откройте файл
- 🛠️ Без установки — никаких команд или пакетов
---

## Возможности

| Функция | Детали |
|---|---|
| 📝 Добавление задач | Название, описание, теги, приоритет, статус, дедлайн |
| 📃 Отображение списка | Карточки задач с чекбоксами для отметки завершения |
| 🔧 Редактирование | Изменение любого поля, смена статуса |
| ❌ Удаление | Удаляйте отдельные задачи или все завершённые одним кликом |
| 📂 Группировка | Переключатели: Все задачи / Только активные / Завершённые |
| ⏰ Напоминания |Просроченные задачи подсвечиваются для внимания |
| 📈 Аналитика | Счётчик и индикатор прогресса в заголовке |
| 🔄 Синхронизация | Автосохранение изменений в локальном хранилище |

## Технический стек

- **HTML5** — семантическая разметка
- **CSS3** — CSS Custom Properties, Grid, Flexbox, анимации
- **JavaScript ES6+** — модульный vanilla JS без фреймворков
- **LocalStorage** — для персистентных данных
- **Font Awesome** — иконки (CDN для минимализма)

##Установка и запуск

1. Загрузите репозиторий:Bash
   ```bash
   https://github.com/sadxwrs/TaskFlow/blob/main/index.html
   ```
2. Откройте файл `index.html` в браузере двойным кликом.  
   Готово — приложение работает.

### Требования

- Современный браузер: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- Интернет-соединение нужно **только** для загрузки шрифтов Google Fonts при первом открытии.  
  Без интернета приложение работает полностью — используются системные шрифты.

## Внутренняя архитектура

```
index.html (скрипт внутри)
├── Модели: Task (структура задачи), TaskManager (управление)
├── Хранение: Загрузка/сохранение в localStorage
├── Операции: Добавление, переключение статуса, редактирование, удаление, очистка
├── Фильтры: По статусу и тексту
├── Вывод: Функция render() для обновления UI
├── Интерфейс: Модал для правок (open/save)
├── Метрики: Расчёт статистики (updateStats)
└── События: Прослушка форм, ввода, клавиш
```
